выполнено в рамках обучающего проекта FSD (Frontend)
---
## Развёртывание
Клонировать
>```git clone https://github.com/shinshilkina/starter-kit-UI.git```

Установка
>```npm i```

Запустить webpack server
>```npm run start:dev```

Запустить production build
>```npm run build```
